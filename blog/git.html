git&github

创建新仓库

切换至目标文件夹 git init

创建密钥对并在github上添加公钥

创建密钥： ssh-keygen -t rsa 对于弹出的输入项一路回车即可 完成后公钥在~/.ssh/id_rsa.pub中

本地仓库连接到远程仓库

git remote add origin <server>

将改动添加到暂存区

git add <filename> git add *

将暂存区提交到本地文件库HEAD

git commit -m "代码提交信息"

提交到远程仓库

git push origin <branch>

从远端更新仓库(自动与本地分支合并)

git pull （origin 远端目标分支:本地目标分支）

当前分支合并至指定分支

git merge <branch>

创建分支

git checkout -b <branch>

切换分支

git checkout <branch>

删除分支

git branch -d <branch>
